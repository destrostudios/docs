# Storage

## Limit default setups
- Set `SystemMaxUse=200M` and `SystemMaxFileSize=50M` in `/etc/systemd/journald.conf` and apply via `service systemd-journald restart`

## Additional storage space
- Additional [storage space](https://helpcenter.netcup.com/en/wiki/server/storagespace) can be booked from netcup and mounted as directory (currently none)
